## Tecnologia

#### Linguagens:

- Kotlin 
- Xml

#### Arquitetura usada:
- MVC

#### Libraries:

## Setup

Para rodar esse projeto, instale-o baixando ou clonando o repositório.

#### Requisitos do sistema 

- Ter um computador
- Android Studio Chipmunk | 2021.2.1
- Minimum sdk v21
- Compile sdk v32
